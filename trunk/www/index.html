Mission

This project is a sub-project of the Scarab Issue Tracker project.

It is a sandbox for the development of software components that

  • are used with the current Scarab application but do not belong to the core application:
  • could be used in future versions of Scarab if they reach maturity.

At the moment and certainly for a while this project does not produce any useful deliverable by itself. So if you are looking for an application, you are at the wrong place, go back to the Scarab home page.

On the other hand, if you are interested in reading/writing code that could shape the future of Scarab, welcome!

Featured sub-projects

Tools & Goodies for the current Scarab application

  • scarab-web-installer : an installation web app for Scarab 0.21 and TRUNK.

Experimental software components

  • scarab-persistent-om : an alternative implementation of the Scarab Object Model + persistence layer using JPA/EJB3/Hibernate instead of Torque (current implementation).
